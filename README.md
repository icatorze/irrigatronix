# irrigatronix
Irrigatronix - Automatic Orchids and Balcony Garden Irrigator

**Based on board NodeMCU 1.0 ESP-12E Module**

## Libraries Dependencies

- AdaFruit Unified Sensor 
- ArduinoJson
- DHT Sensor Library
- MQTT by Jael Gaehwiler
- losant-mqtt-arduino

## Board Dependency

- esp8266 by ESP8266 Community
